# Atharva Agarwal — Personal Portfolio

A modern, interactive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- 🎨 Dark/Light theme toggle
- 💼 Projects & experience showcase
- 📄 Resume download
- 🌟 Animated hero section with cinematic preloader
- 📱 Fully responsive design
- ✉️ Contact section

## Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom animations, glassmorphism, responsive layout
- **Vanilla JavaScript** — Interactive UI, smooth scrolling, theme persistence
